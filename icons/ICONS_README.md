# Extension Icons

This folder should contain the extension icons in the following sizes:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

The SVG files are the editable source artwork. The PNG files are the assets Chrome
loads from `manifest.json`. Keep both sets in sync when changing the brand mark.

## Design Guidelines

The icon uses the VendorPro four-tile mark with the VenDrop purple gradient
(`#667eea` to `#764ba2`). It is drawn separately at each required size so the
mark stays legible in Chrome's 16px toolbar treatment.

When updating the artwork:
- Ensure icons are clear and recognizable at all sizes
- Follow Chrome Web Store icon guidelines
